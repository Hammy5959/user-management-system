# 🧑‍💻 User Management System

A full-stack User Management System with OTP-based authentication and role-based access control.

---

## 🚀 Features

- 🔐 OTP-based user authentication
- 👥 Role-based access control (Admin/User permissions)
- ➕ Create, Read, Update, Delete (CRUD) users
- 🛡️ Secure login system
- 🗄️ Database-driven user management
- ⚙️ Admin panel for managing users and permissions

---

## 🛠️ Tech Stack

**Frontend:**
- Next.js
- React
- TypeScript

**Backend:**
- Flask (Python)
- REST APIs

**Database:**
- SQL / SQLite

---

## 📌 Project Overview

This project is designed to demonstrate a real-world user management system where users must verify their identity using OTP before accessing the system. After verification, users are assigned roles which determine their permissions.

Admins can manage users, assign roles, and control access levels.

---

## 🔐 Authentication Flow

1. User registers / logs in
2. OTP is sent for verification
3. OTP is validated
4. Access granted based on role/permissions
